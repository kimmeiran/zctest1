# 题目图片处理规范

## 一、概述

本文档规定了题目相关图片的统一处理方式，包括图片上传、存储、请求和显示的标准流程。所有涉及题目图片的功能都必须遵循本规范。

## 二、核心原则

### 2.1 图片请求方式

**所有图片请求必须使用 `Authorization` 请求头进行身份验证，不得使用 URL 参数传递 token。**

- ✅ **正确**：使用 `fetchImageAsBlobUrl()` 函数，在请求头中携带 `Authorization: token`
- ❌ **错误**：使用 `getFullImageUrl()` 通过 URL 参数传递 token（已废弃）

### 2.2 图片数据格式

**后端返回的图片数据统一为 JSON 数组字符串格式：**

```json
"[{\"url\":\"/path/to/image.svg\",\"width\":65,\"height\":19},{\"url\":\"/path/to/image2.svg\",\"width\":132,\"height\":24}]"
```

**字段说明：**
- `url`: 图片在服务器上的路径（相对路径或绝对路径）
- `width`: 图片宽度（像素，可选）
- `height`: 图片高度（像素，可选）

### 2.3 图片存储字段

题目相关的图片字段：

| 字段名 | 说明 | 格式 |
|--------|------|------|
| `question_content_images` | 题目内容中的图片（用于替换 `[img]` 占位符） | JSON 数组字符串 |
| `answer_image` | 答案中的图片（用于替换答案中的 `[img]` 占位符） | JSON 数组字符串 |
| `attached_image` | 题目附件图片（无 `[img]` 占位时显示在题目下方） | JSON 数组字符串 |
| `option_images` | 选项图片（单选题/多选题，逗号分隔路径） | CSV 字符串 |

## 三、核心 API 函数

### 3.1 图片上传

**函数：** `uploadImage(file: File)`

**位置：** `src/api/imageUpload.js`

**接口：** `/api/uploadQuestionImage`

**请求方式：** `POST`，`FormData` 包含 `file` 字段

**返回格式：**
```json
{
    "code": 200,
    "msg": "上传成功",
    "data": {
        "url": "/images/1770170438662_04.jpg",
        "width": 1929,
        "height": 1147
    }
}
```

**使用示例：**
```javascript
import { uploadImage } from '@/api/imageUpload.js'

const handleUpload = async (file) => {
    const result = await uploadImage(file)
    if (result.success) {
        const imagePath = result.data  // 图片路径
        const width = result.width     // 图片宽度
        const height = result.height   // 图片高度
        // 处理上传成功后的逻辑
    } else {
        alert(result.message)
    }
}
```

**返回对象结构：**
```javascript
{
    success: true,
    data: "/images/xxx.jpg",  // 图片路径（用于提交给后端）
    url: "/images/xxx.jpg",   // 同 data
    width: 1929,              // 图片宽度
    height: 1147,             // 图片高度
    message: "上传成功"
}
```

### 3.2 图片请求（带 Authorization 请求头）

**函数：** `fetchImageAsBlobUrl(imageUrl: string): Promise<string>`

**位置：** `src/api/imageUpload.js`

**功能：** 使用 `Authorization` 请求头请求图片，返回 blob URL 供 `<img>` 标签使用

**请求方式：**
```javascript
fetch('/api/getImage?fileName=xxx', {
    method: 'GET',
    headers: {
        Authorization: token  // 从 localStorage 获取
    }
})
```

**返回：** `blob:http://localhost:5173/xxx-xxx-xxx`（blob URL）

**使用示例：**
```javascript
import { fetchImageAsBlobUrl } from '@/api/imageUpload.js'

// 单个图片
const blobUrl = await fetchImageAsBlobUrl('/images/xxx.svg')
// 返回: "blob:http://localhost:5173/xxx-xxx-xxx"

// 批量图片
const paths = ['/images/img1.svg', '/images/img2.svg']
const blobUrls = await Promise.all(paths.map(p => fetchImageAsBlobUrl(p)))
// 返回: ["blob:...", "blob:..."]
```

**注意事项：**
- 如果 `imageUrl` 已经是 `blob:` 或 `data:` 或 `http://`/`https://` 开头，直接返回原值
- 失败时返回空字符串 `''`
- 自动处理 SVG 文件的 MIME 类型

### 3.3 替换 `[img]` 占位符

**函数：** `replaceImgPlaceholdersWithAuthAdvanced(content: string, imagesData: string | Array): Promise<string>`

**位置：** `src/api/imageUpload.js`

**功能：** 将文本中的 `[img]` 占位符替换为带鉴权的 `<img>` 标签

**参数：**
- `content`: 包含 `[img]` 占位符的文本
- `imagesData`: 
  - JSON 数组字符串：`"[{\"url\":\"/path/img.svg\",\"width\":65,\"height\":19}]"`
  - 或数组：`[{url: "/path/img.svg", width: 65, height: 19}]`
  - 或 CSV 字符串：`"/path/img1.svg,/path/img2.svg"`

**返回：** HTML 字符串，`[img]` 已替换为 `<img>` 标签

**使用示例：**
```javascript
import { replaceImgPlaceholdersWithAuthAdvanced } from '@/api/imageUpload.js'

// 方式1：JSON 数组字符串（推荐）
const questionContent = "题目内容[img]和[img]"
const imagesJson = "[{\"url\":\"/images/img1.svg\",\"width\":65,\"height\":19},{\"url\":\"/images/img2.svg\",\"width\":132,\"height\":24}]"
const html = await replaceImgPlaceholdersWithAuthAdvanced(questionContent, imagesJson)
// 返回: "题目内容<img src="blob:..." width="65" height="19" ... />和<img src="blob:..." width="132" height="24" ... />"

// 方式2：数组对象
const imagesArray = [
    { url: "/images/img1.svg", width: 65, height: 19 },
    { url: "/images/img2.svg", width: 132, height: 24 }
]
const html2 = await replaceImgPlaceholdersWithAuthAdvanced(questionContent, imagesArray)

// 方式3：CSV 字符串（兼容旧数据）
const imagesCsv = "/images/img1.svg,/images/img2.svg"
const html3 = await replaceImgPlaceholdersWithAuthAdvanced(questionContent, imagesCsv)
```

**生成的 `<img>` 标签特点：**
- `src` 属性为 blob URL（通过 `fetchImageAsBlobUrl` 获取）
- 包含 `width` 和 `height` 属性（如果提供）
- 包含 `style` 属性设置尺寸
- 包含 `class="question-inline-image"` 用于样式控制

### 4.1 题目内容图片（QuestionContent.vue）

**组件位置：** `src/views/question/editQuestion/QuestionContent.vue`

**功能：** 管理题目内容图片和题目附件图片

**图片字段：**
- `form.question_content_images`: 题目内容图片（JSON 数组字符串）
- `form.attached_image`: 题目附件图片（JSON 数组字符串）

**关键函数：**

```javascript
import { uploadImage, fetchImageAsBlobUrl, validateImageFile } from '@/api/imageUpload.js'

// 1. 解析图片字段（从 JSON 字符串或 CSV 转为数组）
const parseImageField = async (val) => {
    if (!val) return []
    let items = []
    if (typeof val === 'string' && val.trim().startsWith('[')) {
        try {
            const arr = JSON.parse(val)
            const paths = arr.map((item) => item.url || item.path || item)
            const urls = await Promise.all(paths.map(p => fetchImageAsBlobUrl(p)))
            items = arr.map((item, index) => ({
                path: item.url || item.path || item,
                url: urls[index] || '',
                width: item.width || null,
                height: item.height || null,
                index
            }))
        } catch {
            // 兼容 CSV 格式
            const paths = val.split(',').filter(p => p.trim()).map(p => p.trim())
            const urls = await Promise.all(paths.map(p => fetchImageAsBlobUrl(p)))
            items = paths.map((path, index) => ({
                path: path,
                url: urls[index] || '',
                width: null,
                height: null,
                index
            }))
        }
    } else if (typeof val === 'string') {
        const paths = val.split(',').filter(p => p.trim()).map(p => p.trim())
        const urls = await Promise.all(paths.map(p => fetchImageAsBlobUrl(p)))
        items = paths.map((path, index) => ({
            path: path,
            url: urls[index] || '',
            width: null,
            height: null,
            index
        }))
    }
    return items
}

// 2. 上传图片后处理
const handleContentImage = async (event) => {
    const file = event.target.files[0]
    if (file) {
        const validation = validateImageFile(file)
        if (!validation.valid) {
            alert(validation.message)
            return
        }
        try {
            const result = await uploadImage(file)
            if (result.success) {
                const blobUrl = await fetchImageAsBlobUrl(result.data)
                contentImages.value.push({
                    path: result.data,        // 原始路径（提交给后端）
                    url: blobUrl,            // blob URL（用于预览）
                    width: result.width || null,
                    height: result.height || null,
                    index: contentImages.value.length
                })
            }
        } catch (error) {
            console.error('上传失败:', error)
        }
    }
}

// 3. 同步到表单字段（watch）
watch(contentImages, (newImages) => {
    const imageArray = newImages.map(img => ({ 
        url: img.path, 
        width: img.width, 
        height: img.height 
    }))
    props.form.question_content_images = JSON.stringify(imageArray)
}, { deep: true })
```

import { replaceImgPlaceholdersWithAuthAdvanced } from '@/api/imageUpload.js'

// 处理题目内容显示
const processQuestionContent = async (question) => {
    if (!question.questionContent || !question.questionContent.includes('[img]')) {
        return question.questionContent
    }

## 五、数据流转流程

### 5.1 上传流程

```
用户选择图片
    ↓
validateImageFile() 验证文件
    ↓
uploadImage() 上传到 /api/uploadQuestionImage
    ↓
后端返回: { url, width, height }
    ↓
fetchImageAsBlobUrl() 获取预览 URL（带 Authorization 请求头）
    ↓
存储到本地数组: { path: url, url: blobUrl, width, height }
    ↓
watch 监听变化，同步到 form 字段: JSON.stringify([{url, width, height}])
```

### 5.2 编辑加载流程

```
打开编辑弹窗
    ↓
从 form.question_content_images 读取 JSON 字符串
    ↓
JSON.parse() 解析为数组
    ↓
批量调用 fetchImageAsBlobUrl() 获取所有图片的 blob URL
    ↓
构建本地数组: [{ path, url: blobUrl, width, height, index }]
    ↓
显示缩略图预览
```

### 5.3 列表/详情显示流程

```
从 API 获取题目数据
    ↓
question_content_images: "[{\"url\":\"/path/img.svg\",\"width\":65,\"height\":19}]"
    ↓
JSON.parse() 解析为数组
    ↓
replaceImgPlaceholdersWithAuthAdvanced() 处理
    ↓
内部批量调用 fetchImageAsBlobUrl() 获取 blob URL
    ↓
替换 [img] 为 <img src="blob:..." width="65" height="19" />
    ↓
v-html 渲染 HTML
```

## 六、注意事项

### 6.1 必须遵循的原则

1. **所有图片请求必须使用 `fetchImageAsBlobUrl()`**，不得使用 `getFullImageUrl()`（已废弃）
2. **图片数据统一使用 JSON 数组字符串格式**，包含 `url`、`width`、`height` 字段
3. **提交给后端时，只提交原始路径**（`path` 字段），不提交 blob URL
4. **缩略图预览使用 blob URL**（`url` 字段），通过 `fetchImageAsBlobUrl()` 获取

### 6.2 兼容性处理

- **向后兼容 CSV 格式**：如果数据是逗号分隔的路径字符串，自动转换为数组格式
- **向后兼容无尺寸数据**：如果 JSON 中没有 `width` 或 `height`，设为 `null`，显示时使用默认尺寸

### 6.3 错误处理

- **上传失败**：显示错误提示，不更新本地数组
- **请求图片失败**：`fetchImageAsBlobUrl()` 返回空字符串，显示占位符或隐藏图片
- **JSON 解析失败**：降级为 CSV 格式解析

### 6.4 性能优化

- **批量请求**：使用 `Promise.all()` 并发请求多张图片
- **缓存 blob URL**：同一路径的图片只请求一次，复用 blob URL
- **懒加载**：列表页面可以延迟加载图片，先显示占位符

## 七、示例代码

### 7.1 完整的图片上传和显示示例

```vue
<template>
    <div>
        <!-- 上传按钮 -->
        <input type="file" @change="handleUpload" accept="image/*" />
        
        <!-- 图片列表 -->
        <div v-for="(img, index) in images" :key="index">
            <img :src="img.url" :style="{ width: img.width + 'px', height: img.height + 'px' }" />
            <span>{{ img.width }}×{{ img.height }}</span>
            <button @click="removeImage(index)">删除</button>
        </div>
    </div>
</template>

<script setup>
import { ref, watch } from 'vue'
import { uploadImage, fetchImageAsBlobUrl, validateImageFile } from '@/api/imageUpload.js'

const props = defineProps({
    form: { type: Object, required: true }
})

const images = ref([])

// 上传图片
const handleUpload = async (event) => {
    const file = event.target.files[0]
    if (!file) return
    
    const validation = validateImageFile(file)
    if (!validation.valid) {
        alert(validation.message)
        return
    }
    
    try {
        const result = await uploadImage(file)
        if (result.success) {
            const blobUrl = await fetchImageAsBlobUrl(result.data)
            images.value.push({
                path: result.data,
                url: blobUrl,
                width: result.width || null,
                height: result.height || null,
                index: images.value.length
            })
        } else {
            alert(result.message)
        }
    } catch (error) {
        console.error('上传失败:', error)
        alert('上传失败，请重试')
    }
}

// 删除图片
const removeImage = (index) => {
    images.value.splice(index, 1)
    images.value.forEach((img, idx) => {
        img.index = idx
    })
}

// 加载已有图片（编辑模式）
const loadImages = async () => {
    const raw = props.form.question_content_images || ''
    if (!raw) return
    
    let items = []
    if (raw.trim().startsWith('[')) {
        try {
            const arr = JSON.parse(raw)
            const paths = arr.map(item => item.url || item.path || item)
            const urls = await Promise.all(paths.map(p => fetchImageAsBlobUrl(p)))
            items = arr.map((item, index) => ({
                path: item.url || item.path || item,
                url: urls[index] || '',
                width: item.width || null,
                height: item.height || null,
                index
            }))
        } catch {
            // 兼容 CSV
            const paths = raw.split(',').filter(p => p.trim())
            const urls = await Promise.all(paths.map(p => fetchImageAsBlobUrl(p)))
            items = paths.map((path, index) => ({
                path: path.trim(),
                url: urls[index] || '',
                width: null,
                height: null,
                index
            }))
        }
    } else {
        const paths = raw.split(',').filter(p => p.trim())
        const urls = await Promise.all(paths.map(p => fetchImageAsBlobUrl(p)))
        items = paths.map((path, index) => ({
            path: path.trim(),
            url: urls[index] || '',
            width: null,
            height: null,
            index
        }))
    }
    images.value = items
}

// 同步到表单
watch(images, (newImages) => {
    const imageArray = newImages.map(img => ({
        url: img.path,
        width: img.width,
        height: img.height
    }))
    props.form.question_content_images = JSON.stringify(imageArray)
}, { deep: true })

// 初始化加载
loadImages()
</script>
```


**计算属性**：
```javascript
const canCreateCareerPaper = computed(() => {
  const user = JSON.parse(sessionStorage.getItem("user") || "{}");
  return user.role === 5 || user.role === 6; // 管理或总管理员
});

const canCreateCompetitionPaper = computed(() => {
  const user = JSON.parse(sessionStorage.getItem("user") || "{}");
  return user.role === 4 || user.role === 5 || user.role === 6; // 考核站管理员、管理或总管理员
});
```

```javascript
}
```

```javascript
{

FormData: {

GET /exam/api/exam/applicants/template
```

3. 批量导入时，如果职工编号不存在，跳过该条记录并记录错误信息
4. 导入完成后，显示导入结果（成功数量、失败数量、失败原因）
